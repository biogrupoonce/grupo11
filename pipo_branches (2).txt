[EN SU BRANCH]
1.- git add *
    -> Para agregar todos los cambios que han hecho al commit que haran

2.- git status
    -> Para que vean todo lo que se va a commitear(revisen que este todo en verde, 
    si hay algo en rojo debe ser porque el archivo se elimino, sino me avisan :v)

3.- git commit -m"Nombre del commit"
    -> para crear su commit

4.- git checkout master
    -> para que se muevan al branch master
